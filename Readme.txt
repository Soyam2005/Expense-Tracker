# TrackGuy Budget App 💰

A minimal and user-friendly web-based Expense Tracker built using HTML, CSS, and JavaScript. Inspired by the Japanese budgeting method, TrackGuy helps users track their income, expenses, bills, and financial progress over time.

---

## ✨ Features

- Dashboard with salary inputs, expense summaries, and visual charts.
- Transactions tab with editable and filterable expense records.
- Budget planner with pie chart distribution and editable bills/expenses.
- Local storage persistence: data stays even after browser reload.
- Input validation and helpful error messages for better user experience.
- Responsive design for desktop and mobile view.

---

## 📁 Files

- `index.html`: Main interface and logic
- `indexstyle.css`: Complete styling
- `README.md`: You’re reading it!

---

## 🚀 How to Use

1. Open `index.html` in any modern web browser.
2. Use the left menu to navigate between Dashboard, Transactions, and Budget Planner.
3. Add income/expenses, filter by category, and view pie charts for budget summary.
4. Your data will be saved automatically using `localStorage`.
